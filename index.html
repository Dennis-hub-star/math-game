Game Instructions

Welcome to the Number Challenge Game! Your objective is to reach the target number by performing mathematical operations using the given set of numbers. The game tests your mental agility and problem-solving skills under pressure.

Game Overview:

  • You’ll be given a target number and a set of numbers.
  • Your goal is to use any combination of the numbers from the set to reach the target number exactly.
  • You can use addition (+), subtraction (-), multiplication (*), or division (/) to achieve the target.

Important Rules:

  1. Single Use: Each number in the set can be used only once in your calculations.
  2. Time Limit:
    • For the Hard Level, you have 6 minutes to complete the challenge.
    • For the Easy Level, you have 3 minutes to complete the challenge.
  3. Operations Allowed: You can use any combination of addition, subtraction, multiplication, and division to reach the target.
  4. No Remainders: If you perform division, the result must be a whole number (no fractions or decimals allowed).
  5. Order of Operations: You can choose the order in which to apply operations. Standard math rules (PEMDAS/BODMAS) apply, so multiplication and division take precedence over addition and subtraction unless you use parentheses to change the order.

How to Play:

  1. Start the Timer: Once you start the level, the timer will begin counting down.
  2. Analyze the Numbers: Look at the target number and the set of numbers provided. Plan your strategy to combine them in the best way.
  3. Calculate Your Answer: Use mental math, pen and paper, or a calculator (if allowed by your game’s rules) to test different combinations.
  4. Submit Your Solution: Once you believe you’ve reached the target number, submit your answer.
